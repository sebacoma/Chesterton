Chesterton Connect™ Accesibilidad Remota

Accede a información crítica de tus bombas desde cualquier lugar y en cualquier momento. La interfaz de usuario de Chesterton Connect está disponible en dos formatos: una aplicación móvil para la instalación y configuración de sensores, y un portal en la nube gestionado por Chesterton que actúa como tu gemelo digital IIoT, monitoreando remotamente tus activos críticos.

Componentes de Chesterton Connect™:

Chesterton Connect Cloud

Servicio basado en suscripción que ofrece acceso remoto y almacenamiento de datos a largo plazo. Permite una visibilidad 24/7 de los sistemas de bombas, proporcionando una vista integral más allá del modo local.

  • Software mantenido por Chesterton
  • Gemelo digital IIoT
  • Indicadores claros y comparaciones de rendimiento
  • Informes automatizados y administración flexible

Chesterton Connect Mobile App

Aplicación disponible para iOS y Android que permite configurar y visualizar datos de sensores, así como realizar cambios de configuración desde tu dispositivo móvil mediante Bluetooth®.

  • Configuración sencilla de sensores
  • Visualización de datos en tiempo real
  • Notificaciones y alertas inmediatas

Chesterton Connect LTE-M Gateway

Gateway Bluetooth® y celular que permite la transferencia automática de datos para bombas y sistemas de sellado monitoreados 24/7 por sensores Chesterton Connect.

  • Soporta hasta 50 dispositivos
  • Conectividad celular LTE-M
  • Instalación sencilla y rápida

Chesterton Connect Universal Gateway

Gateway Bluetooth® con conectividad celular 4G o Ethernet, capaz de manejar hasta 100 dispositivos y con opción de integración SCADA.

  • Soporta hasta 100 dispositivos
  • Conectividad 4G o Ethernet
  • Integración opcional con SCADA
