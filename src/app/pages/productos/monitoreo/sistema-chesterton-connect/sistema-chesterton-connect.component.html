Sistema Chesterton Connect™

El Sistema Chesterton Connect es una solución simplificada de monitoreo de equipos basada en la nube que proporciona visibilidad 24/7 sobre la condición de un equipo. Este monitoreo de equipos en tiempo real puede ayudarte a correlacionar e identificar anomalías temprano para realizar mejoras operativas que aumenten la confiabilidad y minimicen el tiempo de inactividad no planificado. El Sistema Chesterton Connect está diseñado para bombas y sistemas de sellado, pero también se puede utilizar para monitorear vibraciones en otros equipos rotativos, como motores y cajas de engranajes.

Características del Producto

  • Temperatura del proceso

  • Presión del proceso

  • Vibración en 3 ejes (Aceleración, Pico y Velocidad RMS)

  • Temperatura de superficie

  • Batería reemplazable
